# Action Contracts

## Clarify
Return three to five consequential questions, each with a short statement of why it changes the answer.

## Frame
Return prompt type, assumptions, why-now rationale, likely surface, starting lens, and one-sentence problem frame.

## Personas
Compare persona, JTBD, pain, workaround, and priority. Recommend one.

## Options
Return exactly three distinct mechanisms with user value, business value, and complexity or risk.

## Recommend
State the v1, core flow, inclusions, exclusions, and why it is the right first bet.

## Metrics
Return one primary value metric, two supporting metrics, two guardrails, one vanity metric to avoid, and a short causal rationale.

## Risks
Return risk, consequence, mitigation, and guardrail.

## Skeptic
Return the largest gap, likely interviewer pushback, weakest rubric dimension, detected flags, and one-line fixes.

## Summary
Return a 45-second close covering target, problem, v1, tradeoff, primary metric, and guardrail. Add no new ideas.
