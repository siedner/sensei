# Metrics and Guardrails

Derive metrics in this order:

1. User value
2. Observable behavior proving that value
3. Primary metric
4. Supporting diagnostics
5. Guardrails for quality, safety, and unintended consequences

For B2B workflows, useful measures include successful workflow completion, time to complete, self-service rate, manual work avoided, approval latency, setup time, and team activation.

For agentic workflows, pair value with unsafe or blocked action rate, reversal or rollback rate, permission violations, manual intervention, audit completeness, and support burden.

Avoid generic DAU, engagement, NPS, clicks, and inventory counts unless they directly prove the stated value.
