# Demo Flow

1. Run `pnpm dev` and open `http://localhost:3131`.
2. Click **Load sample**.
3. Start the timer and choose **Goal / Context**.
4. Run Clarify, Frame, Personas, Options, Recommend, Metrics, Risks, Skeptic, and Summary.
5. Copy the final summary.
6. Run `/skills`, then `/example-skill outline` in Practice mode to demonstrate extensibility.
7. Build and load the extension, open Notion, select text, and confirm capture.
