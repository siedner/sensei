# Live Interview Playbook

- Clarify only what changes the approach.
- Announce a compact game plan and waypoint between sections.
- Keep speaking while using Sensei; do not let the tool replace judgment.
- Copy only selected blocks into Notion.
- Commit to one persona, one pain, and one first bet.
- If stuck, ask: “What decision must I make next in the answer spine?”
- In the final seven minutes, stop expanding scope and close the loop.
