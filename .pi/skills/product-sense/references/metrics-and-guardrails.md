# Metrics and Guardrails

Start with user value → observable behavior → metric.

For workflows, prefer successful completion, time to complete, self-service rate, manual work avoided, approval latency, error rate, and adoption by the relevant team.

For governed automation, pair value metrics with unsafe action rate, rollback rate, manual intervention, permission violations, audit completeness, and support burden.

Avoid inventory and activity counts that do not prove an outcome.
